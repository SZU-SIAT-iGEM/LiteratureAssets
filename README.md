# LiteratureAssets
作为SZU-SIAT-iGEM2026的文献资料仓库，存放参考的论文，以及阅读学习笔记等。
## 📁 03_文献资料

### 用途说明

存放 **项目相关的参考文献、学习资料、调研报告** 等，方便团队成员共同学习和引用。

### 目录结构

```
03_文献资料/
├── 📁 核心文献/
│   ├── 📄 [作者]_[年份]_[简短标题].pdf
│   └── ...
├── 📁 背景知识/
│   ├── 📁 合成生物学基础/
│   ├── 📁 项目相关技术/
│   └── ...
├── 📁 竞赛资料/
│   ├── 📁 历年优秀项目/
│   ├── 📁 iGEM官方文档/
│   └── ...
├── 📁 调研报告/
│   └── ...
├── 📄 文献阅读清单.md
├── 📄 文献引用格式.md
└── 📄 README.md
```

### 操作指南

#### 上传文献

1. 确认文献与项目相关
2. 重命名文件：`[第一作者姓]_[年份]_[简短标题].pdf`
   - 示例：`Zhang_2024_CRISPR_gene_editing.pdf`
3. 上传至对应分类文件夹
4. 在 `文献阅读清单.md` 中添加条目

#### 文献阅读清单格式

```markdown
## 核心文献

| 文献 | 阅读状态 | 阅读人 | 笔记链接 | 重要程度 |
|------|----------|--------|----------|----------|
| Zhang_2024_xxx.pdf | ✅ 已读 | @张三 | [笔记](链接) | ⭐⭐⭐ |
| Li_2023_xxx.pdf | 🔄 阅读中 | @李四 | - | ⭐⭐ |
| Wang_2022_xxx.pdf | ⏳ 待读 | 未分配 | - | ⭐ |
```

### 注意事项

- **版权问题**：仅上传团队内部使用的文献副本，不要公开分享有版权的内容
- **文件大小**：PDF 文件较大时，可只上传文献信息（标题、DOI、摘要）
- **定期清理**：每月检查，移除不再相关的文献
